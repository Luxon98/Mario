## The game includes a few levels from the original game and also custom worlds made by me.

## :computer:
The game is written in C++. <br>
I have used the following libraries:
- SDL 	     - v. 2.0.10
- SDL_image  - v. 2.0.5
- SDL_mixer  - v. 2.0.4
